# dotfiles
My ArchLinux dotfiles

![alt text](https://raw.githubusercontent.com/toolreaz/dotfiles/master/preview.png)

- Os: Arch Linux
- WM: i3
- Theme: Arc-Dark
- Icons: Arc
- Font: Hack
- Terminal: Termite

## Dependencies

- feh - Wallpaper display
- picom (compton) - Shadows & windows opacity
- rofi - Menu
- i3lock - Lock screen
- polybar - Top bar
